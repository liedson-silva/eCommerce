# E-commerce

## 📋 Descrição

E-commerce moderno, com recursos como cadastro de produtos, carrinho de compras e gerenciamento de pedidos, simulando uma loja virtual completa.

## 🛠️ Tecnologias

- Node.js
- JWT
- Express.js
- React.js
- MongoDB
- Tailwind CSS.
